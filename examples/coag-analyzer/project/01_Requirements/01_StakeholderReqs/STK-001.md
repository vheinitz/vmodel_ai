# Easy sample loading

## Need

A lab technician must be able to load patient samples and start a measurement run
with at most three discrete UI interactions, without consulting a manual.

## Context

High-throughput coagulation labs run hundreds of samples per shift. Each extra step
multiplies operator fatigue and turnaround time. Technicians rotate across instruments
and rarely receive device-specific retraining.

## Acceptance

A technician new to the device can complete a loaded-and-running cycle within
60 seconds in a usability evaluation per IEC 62366.
