# No incorrect results reported

## Need

The system must never report a PT result that has failed internal quality
control checks. Where the system cannot establish result validity, it must
suppress the result and notify the operator.

## Context

Reported coagulation results directly influence anticoagulant dosing decisions.
A silently-incorrect result is more dangerous than a missing result: clinicians
will act on it. ISO 14971 risk analysis identifies result mis-reporting as the
highest-severity hazard for this device.

## Acceptance

For every result reported, an internal QC chain has either passed or been
explicitly bypassed by an authenticated user with documented justification.
