# Manual Start and Stop Control

## Need

The device shall allow the operator to start and stop the mixing process manually at any time. The controls must be immediately accessible, clearly labeled, and operable without special training. Stopping must be possible instantly — not at the end of a fixed cycle — so that the operator can intervene if a problem is observed.

## Context

Lab staff may need to start mixing as soon as a sample is prepared, pause if a container becomes loose, or abort a run if a spill is suspected. A device that only runs pre-programmed cycles without manual override would introduce safety risks and workflow friction.

## Acceptance

Laboratory staff will consider this need met when:
- A single, clearly identifiable control starts the mixing process
- A single, clearly identifiable control stops mixing immediately (not after a delay)
- Controls are reachable without reaching over or near moving parts
- No menu navigation or multi-step sequence is required for basic start/stop
